Prework Study Guide

HTML

  • HTML: Hyper Text Markup.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML is one of the three cornerstone languages in coding
  • HTML is the stucture, or frame, of code.
  • There are three common types of elements: meta relate, content division, and semantic.

CSS

  • CSS: Cascading Style Sheets
  • CSS is the 'style' of an application.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • * rule means that it applies to the entire document.
  • Inline CSS applies to a specific HTML tag.
  • Selector defines an element or attribute to which the rules or declarations apply.
  • Declarations have two important components: the CSS property we want to apply (color, alignment, etc.) and the value of that property (what you want it to look like).

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch name: creates a new branch and switches to it.

JavaScript

  • JavaScript provides logic and interactivity of a webpage.
